# 📊 Samsung Global Sales Analytics & Performance Dashboard

## 🔍 Project Overview

This project delivers an end-to-end business analysis of Samsung’s global sales data using SQL, Python, Machine Learning, and Power BI.

The objective was to analyze revenue performance, identify operational risks, understand customer behavior, and generate data-driven insights to support strategic decision-making.

The dataset contains 15,500+ sales transactions across multiple product categories, customer segments, and sales channels.

---

## 🎯 Business Objectives

- Analyze total revenue and Year-over-Year (YoY) growth
- Identify top-performing products and categories
- Detect high-return-risk products
- Evaluate the impact of discounts on returns
- Understand customer segmentation and channel performance
- Build an executive-level interactive dashboard

---

## 🛠 Tools & Technologies Used

- **SQL** – Data querying, aggregation, window functions
- **Python (Pandas, NumPy, Matplotlib, Seaborn)** – Data cleaning & exploratory data analysis
- **Scikit-learn** – Basic machine learning modeling
- **Power BI** – Interactive dashboard development
- **DAX** – KPI calculations, YoY analysis, return metrics

---

## 📈 Key Insights

- 💰 Total Revenue: **$22.9M**
- 📊 YoY Revenue Growth: **33%**
- 🔁 Overall Return Rate: **9.2%**
- ⚠ High-risk products showed return rates above **12%**
- 🏪 Top revenue channels contributed ~15% of total revenue
- 👥 Individual and Business segments generated the highest revenue
- 📉 Discount sensitivity analysis revealed moderate impact on return rate

---

## 📊 Dashboard Structure

The Power BI dashboard is structured into 4 analytical layers:

### 1️⃣ Executive Overview
- Total Revenue
- YoY Growth
- Units Sold
- Return Rate
- Revenue Trends

### 2️⃣ Product Performance & Category Analysis
- Top Products by Revenue
- Category Contribution
- 5G Revenue Share
- Storage Capacity Analysis

### 3️⃣ Return & Risk Performance Analysis
- Return Rate by Category
- High Return Risk Products
- Discount Sensitivity Analysis
- Revenue Loss %

### 4️⃣ Customer & Sales Channel Intelligence
- Customer Segment Revenue Analysis
- Age Group Contribution
- Channel Performance
- Payment Method Analysis

---

## 🧠 Machine Learning Component

Basic predictive modeling was applied to explore revenue and customer behavior patterns using Python and Scikit-learn.
